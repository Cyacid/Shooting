操作方式:
W.A.S.D-分別對應子彈飛行方向上.左.下.右
方向鍵-主角移動方向
R-重新遊玩
Z-開始遊玩
X-技能-infinity,一段時間內子彈無限，但會把現有子彈歸為一發，消耗mp30
C-技能-iron body,一段時間內不會,遭受到傷害,如果被敵人撞到,敵人即死,消耗mp40
V-技能-bullet time,一段時間內令敵人速度變慢五倍,消耗mp30

遊戲特色-被外星人撞擊時,如果不是被撞到中心點,則會增加分數

分數計算:
	1.一段時間增加1點
	2.擊殺一隻外星人增加50點
	3.擦過外星人,一定時間增加10點
